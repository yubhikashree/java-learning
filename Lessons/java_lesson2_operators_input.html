⚠️ Integer division trap: In Java, 10 / 3 gives 3, not 3.33. When both numbers are int, Java discards the decimal. This is one of the most common bugs in Java — remember it now.
int a = 10, b = 3;

System.out.println(a + b); // 13
System.out.println(a - b); // 7
System.out.println(a * b); // 30
System.out.println(a / b); // 3 ← not 3.33!
System.out.println(a % b); // 1 ← remainder
System.out.println((double)a / b); // 3.3333...
💡 Modulus is more useful than it looks. number % 2 == 0 tells you if a number is even. You'll use this in almost every DSA problem involving even/odd checks.
Part 2 — String Concatenation

Joining text and variables

In Python you might use f-strings: f"Hello {name}". Java uses the + operator to join strings and variables:

String name = "Yubhi";
int age = 17;

System.out.println("Hello, " + name); // Hello, Yubhi
System.out.println("Age: " + age); // Age: 17
System.out.println(name + " is " + age + " years old."); // Yubhi is 17 years old.
⚠️ Watch out: System.out.println("Value: " + 2 + 3) prints Value: 23, not Value: 5. Java reads left to right — once it sees a String, the + becomes concatenation, not addition. Fix: "Value: " + (2 + 3)
Part 3 — Taking User Input

The Scanner class

Python uses input(). Java uses a Scanner object. This is your first encounter with importing a class — a pattern you'll see thousands of times.

import java.util.Scanner; // ← tell Java you need Scanner

public class Main {
public static void main(String[] args) {

Scanner sc = new Scanner(System.in); // create Scanner

System.out.print("Enter your name: ");
String name = sc.nextLine(); // read a String

System.out.print("Enter your age: ");
int age = sc.nextInt(); // read an int

System.out.println("Hello " + name + ", you are " + age);
}
}
1
import java.util.Scanner — Scanner lives in Java's utility library. You must import it before using it, once per file, at the very top.
2
new Scanner(System.in) — creates a Scanner connected to your keyboard input. System.in is the keyboard, just as System.out is the screen.
3
sc.nextLine() reads text. sc.nextInt() reads a whole number. sc.nextDouble() reads a decimal. Match the method to the type you expect.
Putting it together — Grade Calculator

import java.util.Scanner;

public class Main {
public static void main(String[] args) {
Scanner sc = new Scanner(System.in);

System.out.print("Enter your name: ");
String name = sc.nextLine();

System.out.print("Enter marks out of 100: ");
int marks = sc.nextInt();

double percentage = (double) marks / 100 * 100;

System.out.println("--- Result ---");
System.out.println("Student: " + name);
System.out.println("Marks: " + marks + "/100");
System.out.println("Percentage: " + percentage + "%");
}
