## Projects

### Mini Project 1 — Student Management System
A Java console application that manages student records.
- Built with full OOP design
- Features encapsulation with private fields and getters/setters
- Dynamic input for any number of students
- Calculates total, average, grade, and result
- Identifies the topper automatically

**Tech used:** Java, OOP, Arrays, Scanner
